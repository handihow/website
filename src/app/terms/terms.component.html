Beveiliging van gegevens

Er wordt bij apps onderscheid gemaakt tussen de front-end en de back-end. Alle vertrouwelijke gegevens zoals gebruikers en informatie, worden opgeslagen op de back-end. Dit gedeelte wordt beveiligd met security rules.

De front-end code wordt op Github geplaatst zonder alle sleutels (API-keys) die nodig zijn voor de communicatie met de back-end.

Klanten krijgen toegang tot het Github project. Zo is altijd de laatste versie van de broncode beschikbaar.
