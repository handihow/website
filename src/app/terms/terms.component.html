 Open Source De voordelen van open source op een rij

Open source werken heeft de volgende voordelen:

  • Transparantie en continuiteit - het project kan altijd worden voortgezet
  • Veiligheid - de code kan door iedereen worden geinspecteerd
  • Samenwerken en delen - de code kan door iedereen worden gebruikt en bijgewerkt

Veiligheid

Er wordt bij apps onderscheid gemaakt tussen de front-end en de backend. Alle vertrouwelijke gegevens zoals gebruikers en informatie, worden opgeslagen op de back-end. Dit gedeelte wordt niet gedeeld met het publiek.

De front-end code wordt op Github geplaatst zonder alle sleutels (API-keys) die nodig zijn voor de communicatie met de backend.

Zodra een volgende school gebruik wil maken van het systeem, moet deze school dus eerst een eigen database aanmaken en ook eigen sleutels om hiermee te communiceren.
