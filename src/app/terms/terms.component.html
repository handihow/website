 Open Source De voordelen van open source op een rij

Open source heeft de volgende voordelen:

  • Transparantie en continuiteit: het project kan altijd worden voortgezet
  • Veiligheid en kwaliteit: de code kan gemakkelijk worden geinspecteerd
  • Samenwerken en delen: maak met elkaar betere software

Beveiliging van gegevens

Er wordt bij apps onderscheid gemaakt tussen de front-end en de back-end. Alle vertrouwelijke gegevens zoals gebruikers en informatie, worden opgeslagen op de back-end. Dit gedeelte wordt niet gedeeld met het publiek.

De front-end code wordt op Github geplaatst zonder alle sleutels (API-keys) die nodig zijn voor de communicatie met de back-end.

Zodra een andere school gebruik wil maken van het systeem, moet deze school dus eerst een eigen database aanmaken en ook eigen sleutels om hiermee te communiceren.

Het publiceren van de source code op Github betekent dus absoluut niet dat de gegevens van leerlingen ingezien kunnen worden.
